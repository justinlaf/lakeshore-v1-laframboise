# Lakeshore Animal Hospital - Version 1
## Microsoft Stack Build (VS Code + Copilot)

### Design Philosophy
**Aesthetic Direction:** Organic, Warm & Natural
- **Color Palette:** Sage greens, warm terracotta, cream tones
- **Typography:** Crimson Pro (serif) for headings + Work Sans (sans-serif) for body
- **Visual Style:** Soft, approachable, nature-inspired with smooth animations

### Key Features
- **Mobile-First Responsive Design**
- **Smooth scroll animations with Intersection Observer**
- **Gradient backgrounds and organic shapes**
- **Warm color scheme to build trust**
- **Accessibility-focused navigation**

### Technical Stack
- Pure HTML5 / CSS3 / Vanilla JavaScript
- No external frameworks or libraries
- Google Fonts: Crimson Pro & Work Sans
- CSS Grid & Flexbox for layout
- CSS custom properties (variables) for theming

### Color System
```css
--sage: #7a9479 (Primary)
--terracotta: #cb6843 (Accent/CTA)
--cream: #f9f6f0 (Background)
--charcoal: #2a2a2a (Text)
```

### Browser Compatibility
- Chrome/Edge 90+
- Firefox 88+
- Safari 14+
- Responsive breakpoints: 1024px, 768px

### File Structure
```
index.html          (Complete single-file website)
├── Navigation
├── Emergency Banner
├── Hero Section
├── Services Grid
├── About Section
├── Contact Section
└── Footer
```

### Performance Optimizations
- Minimal HTTP requests (single HTML file)
- Optimized font loading with preconnect
- CSS animations using transform/opacity
- Lazy-loading approach for scroll animations

### Deployment Ready
This is a static HTML file ready for immediate deployment to any web server or hosting platform (GitHub Pages, Netlify, Vercel, Azure Static Web Apps, etc.)
